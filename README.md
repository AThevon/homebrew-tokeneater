# homebrew-tokeneater

Homebrew Cask tap for [TokenEater](https://github.com/AThevon/TokenEater) — monitor your Claude AI usage limits from macOS desktop.

## Install

```bash
brew tap AThevon/tokeneater
brew trust AThevon/tokeneater
brew install --cask tokeneater
```

> `brew trust` is required on Homebrew 6.0+, which no longer loads a third-party tap until you trust it.

## Update

```bash
brew upgrade tokeneater
```

## Uninstall

```bash
brew uninstall tokeneater
```
